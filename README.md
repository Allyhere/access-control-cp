## Instalação

1.  Instale as dependências:
    ```bash
    npm install
    ```
2.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
3.  Em outro terminal, inicie a API fake:
    ```bash
    npm run api
    ```
